\doxysubsection{Detailed Description}
Generische JSON-\/\+Dateiverwaltung für Datenobjekte. 

Diese Klasse übernimmt das Speichern und Laden von Datenlisten im JSON-\/\+Format aus einer Datei. Der Pfad zur Datei wird je nach Typ dynamisch gesetzt (z.\+ B. „\+History“, „\+Unit\+Definition“ oder „\+Categories“).
