Detailed Description

ViewModel zur Verwaltung und Darstellung der Umrechnungshistorie.

Diese Klasse stellt eine observable Liste von HistoryEntry -Objekten bereit und lädt Daten entweder aus der Datenbank oder aus einer JSON-Datei – abhängig vom gewählten Modus (Slider).
