Detailed Description

Datenbankkontext für den Einheitenumrechner.

Verwaltet die Verbindung zur Datenbank und definiert die Zuordnung der Entitäten zu Tabellen und Beziehungen. Diese Klasse wird von Entity Framework Core verwendet.
